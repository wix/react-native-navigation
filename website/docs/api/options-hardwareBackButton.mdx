Controls Android hardware back button.

```js
const options = {
  topBar: {
    hardwareBackButton: {},
  },
};
```

### `dismissModalOnPress`

Controls whether the hardware back button should dismiss modal or not.

| Type    | Required | Default |
| ------- | -------- | ------- |
| boolean | No       | true    |
